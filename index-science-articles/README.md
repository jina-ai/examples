The Science Article Corpus</h3>
<p> The folks at Hindawi host a lot of science articles and we want to be able to search it with our cool neural netweorks! Of course, getting the data will mean downloading tonnes of PDF files. No! We can obtain article contents as XML format with the <code>html_crawler()</code> function and feed that into the Flow API. Its pretty simple; first, we obtain the links to the abstracts of the articles and then build the link to the full content with some previous knowledge of the layout of the site.</p>

<p>With the links processed, we proceed to open each XML file and return via the <code>xml_crawler()</code> function, a generator object filled with science articles. Jina's Flow API then concludes the process by consuming the input function dedicated to handling the generator object. That's it. Easy.</p>
